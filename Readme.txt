# Personal Expense Tracker (Java)

A console-based Personal Expense Tracker developed using Java.

## Features
- Add expense
- View expenses
- Search by category
- Calculate total expense
- File storage using text file

## Technologies Used
- Java
- OOP Concepts
- File Handling
- ArrayList
- Exception Handling

## How to Run
1. Compile:
   javac Expense.java ExpenseTracker.java
2. Run:
   java ExpenseTracker
